# pravMotion - An animation plugin for Final Cut pro

PravMotion plugin is a final cut pro plugin and effect built using appleâ€™s fxplug sdk and Apple motion. Create motion graphic animations with ease using this plugin. You can choose various types of easing types for your animation which final cut pro and motion lacks. And The best of all, there are no keyframes. The plugin handles duration set by you. And Using time offset, you can adjust the time for - where the animation should start from, as you desire.

Key Features:

1. Over 20 easing animation types.

2. Duration - control the animation length.

3. Time offset - shift the animation in time to wherever you desire.

4. From and To values - modify the scale, position, rotation, opacity values.

Bonus Tip: Use the effect multiple times and adjust time offset to create one long seamless continuous animation.
